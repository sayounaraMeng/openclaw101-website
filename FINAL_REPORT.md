### 🚀 使用方法

#### 本地打开
```bash
cd /root/.openclaw/workspace/openclaw101-website
# 方式1：直接打开
open index.html

# 方式2：启动本地服务器
python3 -m http.server 8080
# 然后访问 http://localhost:8080
```

#### 功能操作
1. **浏览课程**：点击左侧导航切换天数
2. **查看主题**：点击主题卡片查看详细内容
3. **快速搜索**：按 `/` 键聚焦搜索框
4. **记录进度**：阅读后点击 "Mark as Complete"
5. **代码复制**：点击代码块右上角复制按钮

### 📱 响应式断点

### 🔧 技术栈
- **前端**：HTML5 + Tailwind CSS (CDN)
- **交互**：Vanilla JavaScript (无框架依赖)
- **存储**：LocalStorage (本地进度)
- **内容**：Markdown → HTML 实时渲染
