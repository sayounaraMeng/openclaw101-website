# OpenClaw 101 学习网站

## 项目概述

这是一个为 OpenClaw 101 学习文档设计的精美学习网站，采用类似 Grok 官网的深色科技主题风格。

## 网站目录结构

```
/root/.openclaw/workspace/openclaw101-website/
├── index.html          # 主入口文件
├── css/
│   └── style.css       # 样式文件（深色主题、科技感）
├── js/
│   └── app.js          # 交互逻辑（单页应用）
├── data/
│   └── content.json    # 课程结构和元数据
└── assets/             # 图片等资源（预留）
```

## 主要功能

### 1. 导航系统
- **左侧边栏**：7天课程导航，显示学习进度
- **右侧边栏**：当前页面内容导航（TOC）
- **面包屑导航**：显示当前位置

### 2. 内容展示
- 首页：课程概览、特色介绍、7天学习路径
- 每日页面：显示当天的所有主题
- 主题页面：显示详细的文档内容

### 3. 交互功能
- **代码高亮**：代码块语法高亮
- **复制按钮**：一键复制代码
- **进度追踪**：本地存储学习进度
- **搜索功能**：实时搜索所有主题

### 4. 响应式设计
- 桌面端：三栏布局（导航+内容+TOC）
- 平板端：隐藏右侧TOC
- 移动端：侧边栏可折叠

## 技术栈

- **HTML5**：语义化结构
- **Tailwind CSS**：工具类CSS框架
- **Vanilla JavaScript**：原生JavaScript，无框架依赖
- **LocalStorage**：本地存储学习进度

## 设计风格

- **深色主题**：类似 Grok 官网的深色背景
- **科技感**：渐变色彩、发光效果
- **现代化**：圆角、阴影、过渡动画

## 颜色方案

```css
--bg-primary: #0a0a0f      /* 主背景 */
--bg-secondary: #12121a    /* 侧边栏背景 */
--bg-tertiary: #1a1a25     /* 卡片背景 */
--accent-primary: #6366f1  /* 主色调（靛蓝） */
--accent-secondary: #8b5cf6 /* 次色调（紫色） */
--text-primary: #ffffff    /* 主文字 */
--text-secondary: #a0a0b0  /* 次文字 */
```

## 使用说明

1. 打开 `index.html` 文件即可访问网站
2. 点击左侧导航切换不同天数
3. 点击主题卡片查看详细内容
4. 使用搜索框快速查找内容
5. 阅读完成后点击"Mark as Complete"记录进度

## 特色亮点

1. **单页应用**：无需刷新页面，流畅的用户体验
2. **进度持久化**：刷新页面后学习进度不丢失
3. **代码复制**：所有代码块都支持一键复制
4. **响应式搜索**：支持实时搜索所有主题
5. **移动端友好**：完美适配手机和平板设备
